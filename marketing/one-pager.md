# FieldNotes — One-Pager

## Your workers talk. FieldNotes does the rest.

---

### The Problem

Service business owners have zero real-time visibility into field operations. Workers forget to log issues. Text messages get buried. End-of-day debriefs are a guessing game. Expensive field service software ($100-349/mo) exists — but workers refuse to use it.

**The real competition isn't other software. It's "we just text the boss."**

---

### The Solution

FieldNotes adds one step to your workers' existing routine: a 10-second voice note between stops. 

**Worker:** *"Acme Office: all good, replaced filter, need more filters next time."*

**FieldNotes:**
- Identifies the account
- Logs the service with timestamp
- Flags issues and creates follow-ups
- Updates the supply list
- Adds it to the daily summary

That's it. No apps to download. No forms to fill. No training.

---

### How It Works

```
Worker talks → Telegram/SMS → AI Processing → Service Log
                                              → Action Queue
                                              → Supply List
                                              → Daily Email to Owner
```

---

### Core Features

| Feature | What it does |
|---------|-------------|
| **Voice Input** | Workers dictate notes. No typing required. |
| **Auto Logging** | Every stop timestamped with worker, account, and status |
| **Issue Detection** | "Leaking pipe" → flagged as urgent |
| **Action Queue** | Follow-ups, replacements, supply runs — auto-organized |
| **Daily Summary** | End-of-day email: stops done, issues open, supplies needed |
| **Missed Stop Alerts** | System compares logs vs. schedule, flags gaps |
| **Multi-Worker** | Each worker's notes tagged separately |
